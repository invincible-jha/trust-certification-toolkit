## Quick Rules

- Read `FIRE_LINE.md` before writing any code
- All source files must carry the SPDX license header
- Python: type hints on every function signature, Pydantic v2 for validation
- TypeScript: strict mode, no `any`, Zod for runtime validation
- Conventional commits: `feat:`, `fix:`, `refactor:`, `docs:`, `test:`, `chore:`
- No automatic trust progression, behavioral scoring, or external communication
